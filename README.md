# Personal Website Template

This is a static personal website template for the optional CV homework bonus.
It is designed for GitHub Pages and does not require any build tools.

## Edit These First

- `index.html`
  - Replace `Your Name`.
  - Replace email, GitHub URL, education, awards, projects, skills, and research interests.
  - If you have a CV PDF, place it at `assets/your_name_CV.pdf` or update the CV links.
- `styles.css`
  - Optional: adjust colors in the `:root` block.
- `assets/hero-study-desk.png`
  - Optional: replace it with your own photo or a more personal academic image.

## Publish With GitHub Pages

Deadline for the optional bonus: July 17, 23:59.

Recommended route:

1. Create a public GitHub repository named `<your-github-username>.github.io`.
2. Upload everything inside this `personal_website` folder to the repository root.
3. Commit the files to the `main` branch.
4. Open `https://<your-github-username>.github.io` after GitHub Pages finishes publishing.

Alternative route:

1. Create any public repository, for example `personal-website`.
2. Upload this folder's contents.
3. Go to repository `Settings -> Pages`.
4. Choose `Deploy from a branch`, then select `main` and `/root`.
5. Use the Pages URL shown by GitHub.

## Checklist

- The home page opens without local build tools.
- The CV link works, or the link is removed.
- The page includes self-introduction, education, awards, skills, and research interests.
- The site is publicly accessible before the bonus deadline.
